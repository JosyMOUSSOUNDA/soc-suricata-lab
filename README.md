# SOC Suricata Lab

This project demonstrates the deployment of a small Security Operations Center (SOC) lab using Suricata to detect network attacks and analyze suspicious traffic.

## Project Objectives

The goal of this project is to simulate how a SOC environment detects and analyzes malicious network activity.

Main objectives:

- Deploy an Intrusion Detection System (IDS)
- Monitor network traffic
- Detect simulated cyber attacks
- Analyze alerts generated by the IDS

## Lab Architecture

The lab environment is composed of several virtual machines:

Attacker machine → Kali Linux  
Target machine → Linux server  
Monitoring system → Suricata IDS

All machines are deployed in a virtual environment using VirtualBox.

## Tools Used

- Suricata (Intrusion Detection System)
- Wireshark (Network traffic analysis)
- Kali Linux (attack simulation)
- VirtualBox (virtualization)

## Attacks Simulated

During the lab the following attacks were simulated:

- SYN Flood
- Port scanning
- Suspicious network traffic patterns

These attacks were detected by Suricata and analyzed using Wireshark.

## Detection and Analysis

Suricata generates alerts based on network signatures. These alerts are then analyzed to understand the attack behavior and identify potential threats.

Example detections:

- abnormal TCP traffic
- port scanning detection
- suspicious packet patterns
